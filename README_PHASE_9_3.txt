PHASE 9.3 — EDIT BUDGET

Settings now includes a collapsible Edit Budget menu.

Features:
- Shows every expense category for the selected month.
- Each monthly budget can be edited.
- Save Budgets updates:
  Budget
  Balance = Budget - Spent
  Home selected-category card
  All Categories
  Quick Categories
  Dashboard category rows
- The next month remembers the edited budgets because Create Next Month copies the latest monthly sheet.

INSTALL
1. Replace Apps Script Code.gs with backend/Code.gs.
2. Save.
3. Deploy > Manage deployments > Edit > New version > Deploy.
4. Replace repository files with this package.
5. Commit:
   Phase 9.3 edit budget
6. Push origin.
