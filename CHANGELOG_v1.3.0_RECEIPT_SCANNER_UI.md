# v1.3.0 — Receipt Scanner UI

## Added
- New Scan Receipt button on Home.
- Separate Receipt Scanner view.
- Camera capture and photo-library selection.
- Receipt image preview with rotation control.
- Crop placeholder for the upcoming OCR engine.
- Review card that inherits the active Home user.
- Safe prototype Save flow that does not create an expense.

## Scope
This is a frontend-only UI/UX release. OCR, merchant detection, amount extraction, category suggestions and backend saving are intentionally not included.
