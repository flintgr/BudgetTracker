PHASE 9.1 — CREATE NEXT MONTH

Month Management now includes:
- Create Next Month
- Clear Month Data
- Delete Month

Create Next Month:
- Uses the latest monthly sheet as the source.
- Creates the next calendar month.
- Keeps Budgets.
- Keeps Income Sources.
- Starts Spent at 0.
- Restores Balance from Budget.
- Selects the newly created month automatically.
- Returns to Home after creation.

INSTALL
1. Replace Apps Script Code.gs with backend/Code.gs.
2. Save.
3. Deploy > Manage deployments > Edit > New version > Deploy.
4. Replace repository files with this package.
5. Commit:
   Phase 9.1 create next month
6. Push origin.
